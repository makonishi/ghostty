# Ghostty Configuration

Vim風キーバインドを中心としたGhostty設定。キーボードだけで全操作を完結できる。

## Appearance

| 設定 | 値 |
|------|-----|
| フォント | JetBrainsMono Nerd Font Mono |
| フォントサイズ | 13.7 |
| テーマ | Tomorrow Night |
| カーソル | block |
| カラースペース | display-p3 |

## キーバインド一覧

リーダーキーは `Ctrl+Q`。

### 常時有効

#### スクロール

| キー | 動作 |
|------|------|
| `Ctrl+D` | 半ページ下スクロール |
| `Ctrl+U` | 半ページ上スクロール |

#### ペイン操作

| キー | 動作 |
|------|------|
| `Ctrl+Q > h/j/k/l` | ペイン移動（左/下/上/右） |
| `Ctrl+Q > v` | 縦分割（右に新規） |
| `Ctrl+Q > s` | 横分割（下に新規） |
| `Ctrl+Q > x` | ペインを閉じる |
| `Ctrl+Q > z` | ペインズーム切替 |
| `Ctrl+Q > =` | ペインサイズ均等化 |

#### ペインリサイズ

| キー | 動作 |
|------|------|
| `Ctrl+,` | 左に広げる |
| `Ctrl+.` | 右に広げる |
| `Ctrl+;` | 下に広げる |
| `Ctrl+'` | 上に広げる |

#### タブ操作

| キー | 動作 |
|------|------|
| `Ctrl+Q > t` | 新規タブ |
| `Ctrl+Q > n` | 次のタブ |
| `Ctrl+Q > p` | 前のタブ |
| `Ctrl+Q > 1-5` | タブ番号で移動 |

### Vimモード

`Ctrl+Q > Escape` で入る。モード中はウィンドウにインジケーターが表示される。

`catch_all=ignore` により、バインドされていないキーはすべて無視される（ターミナルに送信されない）。

#### スクロール

| キー | 動作 |
|------|------|
| `j` / `k` | 3行ずつ下/上スクロール |
| `Ctrl+D` / `Ctrl+U` | 半ページ下/上 |
| `Ctrl+F` / `Ctrl+B` | 1ページ下/上 |
| `g` | 最上部へ |
| `G` | 最下部へ |

#### プロンプトジャンプ

シェルインテグレーション（Ghostty built-in）が必要。

| キー | 動作 |
|------|------|
| `[` | 前のプロンプトへ |
| `]` | 次のプロンプトへ |

#### 選択

既存の選択がある場合のみ動作する（マウスで範囲選択後に調整する形）。

| キー | 動作 |
|------|------|
| `H` / `J` / `K` / `L` | 選択範囲を左/下/上/右に拡張 |
| `0` | 行頭まで選択 |
| `$` | 行末まで選択 |

#### クリップボード

| キー | 動作 |
|------|------|
| `y` | コピー |
| `p` | ペースト |

#### 検索

| キー | 動作 |
|------|------|
| `/` | 検索開始 |
| `n` | 次の検索結果 |
| `N` | 前の検索結果 |

#### タブ・ペイン操作（Vimモード内）

| キー | 動作 |
|------|------|
| `t` | 新規タブ |
| `Ctrl+N` / `Ctrl+P` | 次/前のタブ |
| `1-5` | タブ番号で移動 |
| `Ctrl+Q > h/j/k/l` | ペイン移動 |
| `Ctrl+Q > v/s` | 縦/横分割 |
| `Ctrl+Q > x` | ペインを閉じる |
| `Ctrl+Q > z` | ペインズーム |
| `Ctrl+Q > =` | ペイン均等化 |

#### モード終了

| キー | 動作 |
|------|------|
| `Escape` | Vimモード終了 |
| `i` | Vimモード終了（insert感覚） |
